Detalhes do projeto

Este projeto foi desenvolvido como parte da disciplina de Projeto de Sistema de Software. O objetivo deste documento é apresentar o aplicativo de ônibus urbano SGBus. Inicialmente concebido para atender à população de São Mateus, o aplicativo traz modernização para a empresa responsável pelo transporte público, além de oferecer facilidade e portabilidade aos usuários.
